# Technical Context

## Core Technologies
- **HTML5:** For the structure of the web application.
- **CSS3:** For styling the user interface.
- **Vanilla JavaScript (ES6):** For all client-side logic, including DOM manipulation, event handling, and dynamic calculations. No external frameworks or libraries are used.

## Development Environment
- **Editor:** Visual Studio Code
- **Shell:** Windows PowerShell 5.1
- **Local Server:** Attempts to use `python -m http.server` and `npx http-server` failed. The recommended approach for local development is using the **"Live Preview"** VS Code extension.

## Version Control & Hosting
- **Version Control:** Git
- **Repository:** GitHub (`https://github.com/MorgB-C/ebayAuktion`)
- **Hosting:** GitHub Pages, configured to deploy from the `main` branch.
